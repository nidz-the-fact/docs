## thirdweb

[**Universal Bridge**](https://thirdweb.com/connect/universal-bridge) is a comprehensive Web3 payment solution to help you monetize any app or game.

With Universal Bridge, your users can **onramp, bridge, and swap** on any EVM chain — with any EVM token or fiat — thanks to its **automatic cross-chain routing**.

Plus, developers can **earn from day one** using the **fee-sharing mechanism** and **easy implementation**.

**Supported Networks**
- [Superseed Mainnet](https://thirdweb.com/bridge?chainId=5330)

### Universal Bridge Features

- Let users pay for assets in any EVM token on Superseed right in your app
- Automatic cross-chain routing for seamless transactions
- Earn from day one with fee-sharing mechanism
- Access ready-made UI component for easy implementation

[Learn more in the full documentation on thirdweb’s Universal Bridge](https://portal.thirdweb.com/connect/pay/overview)

### Get Started with thirdweb Universal Bridge

[How to get started with Universal Bridge](https://portal.thirdweb.com/connect/pay/get-started)

### See Universal Bridge in Action
Want to see how Universal Bridge works? Check it out under the hood

```jsx
import { createThirdwebClient } from "thirdweb";
import { PayEmbed } from "thirdweb/react";

import { createWallet } from "thirdweb/wallets";
import { defineChain } from "thirdweb";

const client = createThirdwebClient({
  clientId: "....",
});

function Example() {
  return (
    <PayEmbed
      client={client}
      payOptions={{
        mode: "fund_wallet",
        prefillBuy: {
          chain: defineChain(5330),
          amount: "0.01",
        },
      }}
    />
  );
}

```

## [Hyperlane Nexus](https://www.usenexus.org/?origin=ethereum&destination=superseed)

Hyperlane is a permissionless interoperability protocol for cross-chain communication across different blockchain environments.
It enables message passing and asset transfers across different chains without relying on centralized intermediaries or requiring any permissions.

**To find out more please check out the [Hyperlane Documentation](https://docs.hyperlane.xyz/).**

## [Layerswap](https://layerswap.io/app)

Layerswap is the reliable solution for transferring crypto assets across networks in a matter of minutes.

The app enables frictionless cross-chain transfers across 35+ blockchains as well as direct transfers between chains and 15+ exchanges. See the complete list of available assets, chains and CEXes.

**To find out more please check out the [Layerswap Documentation](https://docs.layerswap.io/user-docs).**

## [Relay](https://www.relay.link/bridge/superseed)

Relay is a cross-chain payments system that enables instant, low-cost bridging and cross-chain execution. It works by connecting users with relayers.

**To find out more please check out the [Relay Documentation](https://docs.relay.link/what-is-relay).**
